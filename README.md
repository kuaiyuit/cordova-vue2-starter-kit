## Build Setup

``` bash
# install dependencies
$ npm install
$ npm install -g cordova
$ cordova prepare

# serve with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build

# build android
$ cordova build android

# run in android
$ cordova run android

#build ios
$ cordova build ios

# run in ios
$ cordova run ios
```
